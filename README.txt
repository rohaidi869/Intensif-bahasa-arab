LATIHAN INTERAKTIF الضمائر (KATA GANTI NAMA)

CARA MENGGUNAKAN:

1. MUAT TURUN DAN BUKA FILE
   - Muat turun fail "latihan-damair.html"
   - Klik kanan pada file → "Open with" → Pilih pelayar web (Chrome disyorkan)

2. LENGKAPKAN MAKLUMAT DIRI
   - Isi nama penuh, ID pelajar, dan kod tugasan (jika ada)

3. JAWAB SOALAN
   - Terdapat 4 bahagian latihan:
     * المفردات (Kosa Kata)
     * رتب الكلمات (Susun Ayat)
     * أكمل الحوار (Lengkapkan Dialog)
     * اختر الإجابة (Soalan Pilihan)
   - Jawab semua soalan dalam setiap bahagian
   - Gunakan butang "تحقق من الإجابة" untuk menyemak jawapan

4. HANTAR HASIL KERJA
   - Setelah SEMUA bahagian selesai, klik "Jana Fail Penghantaran"
   - File JSON akan dimuat turun secara automatik
   - Muat naik file JSON tersebut ke Google Classroom

5. FUNGSI TAMBAHAN
   - "Simpan Kemajuan" - untuk menyimpan dan meneruskan kemudian
   - "Muat Semula Kemajuan" - untuk memuatkan kerja yang disimpan
   - Butang audio untuk mendengar sebutan Arab

NOTA PENTING:
- Pastikan anda menggunakan pelayar web moden (Chrome, Firefox, Edge)
- File mesti dibuka melalui pelayar web, bukan editor teks
- Semua jawapan akan disimpan dalam file JSON untuk penilaian
